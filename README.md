# Prehtagonist - Web3 Professional Portfolio

A sleek, modern single-page website for Prehtagonist, showcasing Web3 expertise and professional presence.

## Features

- **Modern Dark Theme**: Clean black and deep gray color scheme with subtle blue accents
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices
- **Smooth Animations**: Elegant fade-in and hover effects
- **Professional Layout**: Minimalist design focused on content and usability
- **Social Media Integration**: Direct links to X, Telegram, Discord, and WhatsApp
- **Performance Optimized**: Fast loading with modern CSS and JavaScript

## Files Included

- `index.html` - Main HTML structure
- `styles.css` - Complete styling with dark theme and animations
- `script.js` - JavaScript for interactive elements and animations
- `open-website.bat` - Windows batch file to open the website

## How to View

### Option 1: Direct File Opening
Double-click on `open-website.bat` or open `index.html` directly in your browser.

### Option 2: Local Server (Recommended)
If you have Python installed:
```bash
python -m http.server 8000
```

If you have Node.js installed:
```bash
npx serve . -p 8000
```

Then visit `http://localhost:8000` in your browser.

## Customization

To customize the content:
1. Edit `index.html` to change the headline, bio, and social links
2. Modify `styles.css` to adjust colors, fonts, or spacing
3. Update `script.js` for additional interactive features
